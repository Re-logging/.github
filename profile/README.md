# 서비스 소개
리로깅은 플로거들에게 다양한 환경뉴스 및 지자체 플로깅 정보를 제공하고, 플로거들간 커뮤니티가 활성화될 수 있도록 돕는 서비스입니다.

# 주요 기능
- 지자체 플로깅 정보 제공
- 플로깅 모임 생성 및 참여
- 환경 뉴스 제공
- 소셜 로그인 (Google, Kakao)
